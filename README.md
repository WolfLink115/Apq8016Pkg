# Apq8016Pkg
### Project Silicium WIP Silicon Port for APQ8016 SoCs. Vibe-Coded with Kilo Code.

## Status
### Non-Booting Yet.

## Setting up your Environment
### Tested on Linux Mint 23.2

### First, clone Project Silicium and Silicon Source Code:

```
mkdir -p ~/silicium && cd ~/silicium
```

```
git clone https://github.com/Project-Silicium/Mu-Silicium.git --recursive
git clone https://github.com/WolfLink115/Apq8016Pkg Mu-Silicium/Silicon/Qualcomm/Apq8016Pkg
```

```
./setup_env.sh -p apt
```
